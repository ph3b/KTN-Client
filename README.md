# KTN-Client
[![Build Status](https://travis-ci.org/ph3b/KTN-Client.svg?branch=master)](https://travis-ci.org/ph3b/KTN-Client)

Simple chat cli to use with KTN-ChatServer ( https://github.com/ph3b/KTN-ChatServer/ )

# Install and run
    $ npm install
    $ node client.js
    
# Test
    $ npm test
